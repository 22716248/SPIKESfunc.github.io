SPIKEStudy

Drug Dose Responses Interactive Learning Tool

Dose-Response Visualiser

This tool can be used in order to learn the effects of how drugs bind to receptors using SPIKES analysis to qualitatively view changes to graphs, quantitatively alter them, and an interactive test tool to check your knowledge.

Investigate the patterns!

Schild Plot Generator

Quantitative analysis of the effect of competitive antagonists on agonist-induced effects is a powerful method for identifying the receptors that mediate agonist-induced effects.

See it at work!

Schild Plot Analysis Quiz

Quiz tool that students can use to develop competencies in interpreting Schild analyses.
